SSML Spec Features
==================
Prosody & Style
---------------
* Allow options hash for setting attributes in initializers
* Restrict child element types for `<voice/>`, `<emphasis/>`, `<break/>`, `<prosody/>`
* Documentation

Document Structure
------------------
* `<p/>` and `<s/>`
* `<say-as/>`
* `<phoneme/>`
* `<sub/>`
* `<lexicon/>`
* `<meta/>` and `<metadata/>`

Misc
----
* `<audio/>`
* `<mark/>`
* `<desc/>`
